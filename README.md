# Theme: Scorsese

## Installation

Visit the [Fork CMS knowledge base](http://fork-cms.com/knowledge-base) to learn how to install a theme. To download the zip-package go to the [extension page](http://www.fork-cms.com/extensions/detail/scorc) of the theme on fork-cms.com.
